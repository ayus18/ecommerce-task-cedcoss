\# SpeedDeliver — Shipment Tracking Page



A shipment tracking UI built for the Cedcoss Technologies Round 2 assignment.



\## What it does

Enter an order ID to see a real-time style shipment timeline with animated steps, live map, and ETA.



\## How to run

Just open index.html in any browser. No installation needed.



\## Sample Order IDs

\- SD-10482 (Out for Delivery)

\- SD-20931 (In Transit)



\## Tools used

\- HTML, CSS, JavaScript

\- Leaflet.js (for map)

\- Google Fonts (Syne + DM Sans)

\- AI assistance (Claude) for code generation



\## Project Structure

\- index.html — complete tracking page (single file)
